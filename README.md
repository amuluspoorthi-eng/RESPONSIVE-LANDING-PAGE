# Responsive Landing Page

## 📌 Overview

This project is a modern and responsive landing page developed using **HTML**, **CSS**, and **JavaScript**. It features a fixed navigation bar with interactive hover effects and dynamic styling that changes when the user scrolls the page.

The design is fully responsive and works seamlessly across desktop, tablet, and mobile devices.

---

## 🚀 Features

- Responsive Landing Page
- Fixed Navigation Bar
- Navigation Bar Changes Color on Scroll
- Hover Animation for Navigation Links
- Mobile-Friendly Navigation Menu
- Smooth Scrolling
- Modern UI Design
- Clean and Well-Structured Code

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🎯 Project Objective

The objective of this project is to create an interactive navigation menu that:

- Remains fixed at the top of the page.
- Changes its appearance when scrolling.
- Includes hover effects on menu items.
- Provides a responsive layout for different screen sizes.
